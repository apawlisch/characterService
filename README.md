# Dungeons and Dragons Character Service
This service keeps track of and manages adventurers' health, speed, basic traits, initiative, and armor class.

## Features
* Get Adventurer
* Add to Current Health
* Remove from Current Health
* Restore Current Health to Max Health
* Set Max Health
* Reset Max Health (*Calculate Based on Traits*)
* Set Traits
* Set Speed
* Reset Speed (*Calculate Based on Traits*)
* Set Initiative
* Set Armor Class
* Create Adventurer
    * Create with Name
    * Create with Name and Traits
    * Create with Name and Armor Class
    * Create with Name and Traits and Armor Class
* Delete Adventurer


**This service was built for practicing code, it might not be very useful for actually running a D&D game**
